Core Initialization Tests\n- `testNpcSimulatorInitialization` - Verifies default simulator state\n- `testEmptySimulation` - Validates empty simulation behavior\n\n### 2. NPC Lifecycle Tests\n- `testNpcCreationAndStateTransition` - Tests NPC creation and state preservation\n- `testNpcInteractionSystem` - Validates NPC interaction mechanics\n- `testResourceCleanup` - Ensures proper memory management\n\n### 3. Behavioral Pattern Tests\n- `testBehavioralPatternEvaluation` - Tests behavior engine with mocked dependencies\n- `testDeterministicBehavior` - Validates consistent output for same inputs\n- `testConcurrentSimulationScenarios` - Verifies parallel action processing\n\n### 4. Simulation Mechanics Tests\n- `testSimulationStepProgression` - Tests turn-based simulation advancement\n- `testConfigurationLoading` - Validates configuration system\n\n### 5. Error Handling Tests\n- `testInvalidInputHandling` - Verifies proper exception handling\n\n## Running Tests\n\n### Command Line\n
